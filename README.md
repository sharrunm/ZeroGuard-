# CyberCrimeAI - Phase 2 (Home Page UI Design)

Welcome to the **CyberCrimeAI** Home Page! This is the frontend-only Phase 2 implementation designed with modern, state-of-the-art UI elements, custom dark/light theme accents, and dynamic interaction states.

## 🎨 Theme Specs Used
* **Primary Color**: `#0F4C81` (Deep Blue)
* **Secondary Color**: `#1E3A5F`
* **Accent Color**: `#2E7D32` (Green)
* **Danger Color**: `#C62828` (Red)
* **Background Color**: `#F5F7FA`
* **Cards**: `White`

## 🛡️ Completed Features
1. **Interactive AI Chat Sandbox**: Watch the AI detect a sample Banking Fraud incident in real-time, change risk levels dynamically, present target questions, and suggest actionable checklists.
2. **15 specialized Cyber Crime Categories**: Fully featured grid layout representing all categories from the project roadmap, with responsive live-search capabilities.
3. **Emergency Numbers Portal**: One-tap emergency helpline calls (1930 / 112 / 1091).
4. **Interactive Stats Counter**: Numbers animate and count up automatically as you scroll down the page.
5. **Modern Navigation Layout**: Fully responsive sticky navigation bar with a mobile drawer menu and auto scroll-indicator accents.
